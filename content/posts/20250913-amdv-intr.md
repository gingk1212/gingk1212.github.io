AMD 向けハイパーバイザを自作した際にゲストの外部割り込みをどのように実装したかを紹介する。あくまで一例。なお、設計は例によって Writing Hypervisor in Zig ([割り込みの注入](https://hv.smallkirby.com/vmm/intr_injection)) に沿っている。リポジトリはこちら: [gingk1212/ymirc](https://github.com/gingk1212/ymirc)

前提

- PIC を使用 (APIC は未使用)
- ホストとゲストの両者が外部割り込みを受け取ることができるようにする

設計

- ゲスト実行中に外部割り込みが発生した場合は #VMEXIT してホストの割り込みハンドラを実行させる
- 割り込みハンドラ処理の中でゲストに注入すべき IRQ 番号を保存しておく
- VMRUN を実行する前に割り込みに関する情報を VMCB に書き込んでおくことで外部割り込みをゲストに注入する

それでは、実装について説明していく。まず、VMCB の初期設定としては以下の項目をセットする。INTERCEPT_INTR は

```c
  vmcb->intercept_intr = 1;
  vmcb->v_ign_tpr = 1;
  vmcb->v_intr_masking = 1;
```

```c
  vcpu->vmcb->v_irq = 1;
  vcpu->vmcb->v_intr_vector = is_primary(i)
                                  ? delta(i) + guest_state->primary_base
                                  : delta(i) + guest_state->secondary_base;
```

```c
  case SVM_EXIT_CODE_INTR:
    // If a physical interrupt occurs before the virtual interrupt is
    // consumed, set the virtual interrupt back to the pending IRQ.
    if (vcpu->vmcb->v_irq == 1) {
      vcpu->pending_irq |= tobit_16(vcpu->last_injected_irq);
    }

    // Consume the interrupt by YmirC. At the same time, interrupt subscriber
    // sets the peinding IRQ.
    stgi();
    clgi();

    // Give the external interrupt to guest.
    inject_ext_intr(vcpu);
    break;
```
